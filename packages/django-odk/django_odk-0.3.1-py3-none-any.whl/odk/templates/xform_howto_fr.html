Comment créer un formulaire XForm?

  1. Construire un fichier Excel avec 3 onglets: 'survey', 'choices', 'settings' Cfr. documentation ci-dessous
  2. Soumettre ce fichier sur https://getodk.org/xlsform/
  3. Vérifier que le formulaire fonctionne bien en cliquant sur 'Preview in browser'
  4. Télécharger le fichier XForm (fichier XML) sur votre ordinateur
  5. Charger ensuite ce fichier xml en cliquant sur 'Choose file'
  6. Cliquer sur 'Enregistrer'
